# Ashley Townley Portfolio — Static HTML

This version uses plain static pages instead of React routing.

Upload all files and the `assets` folder to GitHub.

No build settings are required on Vercel.

Recommended Vercel settings:
- Framework preset: Other
- Build command: leave blank
- Output directory: leave blank

Pages:
- index.html
- work.html
- eras.html
- systems.html
- pet-projects.html
- about.html
- contact.html
